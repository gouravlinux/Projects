# E_CHAYAN Project 

Website Link: https://electronicchayan.onrender.com/

Youtube Tutorial Link: https://youtu.be/v1sn2jMZMkM?si=ytJLzmTT8rAy50hv

 

**Tech Stack:** Django (Python), PostgreSQL, Bootstrap 5, Render (Cloud).

# **Key Features & Functionality:**

- **Secure Authentication:** Implements Email-based OTP verification and strong password policies for both Voters and Candidates.
    
- **Role-Based Dashboards:** Separate interfaces for Voters (to view eligible elections and cast votes) and Candidates (to manage their campaign profile and party affiliation).
    
- **Election Logic:** Dynamic filtering of elections based on the user's State, with strict backend validation to prevent double-voting.
    
- **Real-Time Analytics:** Homepage visualization of live voter turnout percentages without revealing partial results.
    
- **Cloud Deployment:** Fully deployed on Render with WhiteNoise for static file serving and a production-ready PostgreSQL database.
